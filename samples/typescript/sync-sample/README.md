# TypeScript Sync Sample

This sample demonstrates `async/await` usage to perform sequential operations resembling a synchronous flow using the `@aeron-cache/embedded-client`.

## Prerequisites

- Node.js (v14+)
- NPM

## Running the Sample

1. Install dependencies:
   ```bash
   npm install
   ```

2. Build the TypeScript code:
   ```bash
   npm run build
   ```

3. Run the sample:
   ```bash
   npm start -- http://localhost:8080
   ```

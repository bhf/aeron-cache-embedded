# Java Sync Sample

This sample demonstrates how to use the synchronous API of the `aeron-cache-embedded-client`.

## Prerequisites

- Java 11 or higher
- Gradle

## Running the Sample

1. Navigate to this directory.
2. Run the application using Gradle:

```bash
gradle run --args="http://localhost:8080"
```

Replace `http://localhost:8080` with the actual URL of your Aeron Cache server if different.
